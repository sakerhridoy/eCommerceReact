# Exclusive E-Commerce React Project

A modern, fully responsive **e-commerce website** built with **React**, **Tailwind CSS**, and **React Router**, featuring a shop, product details, order tracking, and wishlist functionality. This project is inspired by modern online shopping platforms and includes reusable components, context-based state management, and user-friendly UI.

##  Live: 
#### https://full-ecommece.netlify.app/

##  Features

- **Responsive UI**: Works seamlessly on mobile, tablet, and desktop.
- **Shop & Category Pages**: Display products dynamically fetched from a dummy API.
- **Product Details**: 
  - Image gallery with selectable thumbnails.
  - Color and size selection.
  - Quantity selector with add-to-cart functionality.
- **Cart & Wishlist**:
  - Add/remove products.
  - Context-based global state management.
- **Order Tracking**: 
  - Users can track orders by Order ID and Email/Phone.
  - Dynamic order status and timeline display.
- **Order Success Page**: 
  - Order summary card.
  - Track your order button.
  - Continue shopping button.
- **Related Products**: Shows related items on product detail page with quick add-to-cart/wishlist buttons.
- **SweetAlert2 Toasts**: Friendly notifications for adding items to cart or wishlist.
- **SEO-Friendly Breadcrumbs** for better navigation.
